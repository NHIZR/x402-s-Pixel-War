## Get Started in 5 Minutes

Follow these simple steps to start conquering pixels!

<Steps>
  <Step title="Install a Solana Wallet">
    Choose one of these popular wallets:

    <Tabs>
      <Tab title="Phantom (Recommended)">
        1. Visit [phantom.app](https://phantom.app/)
        2. Click "Download" for your browser
        3. Install the extension
        4. Create a new wallet
        5. **Save your seed phrase securely!**
      </Tab>
      <Tab title="Solflare">
        1. Visit [solflare.com](https://solflare.com/)
        2. Download browser extension
        3. Set up your wallet
        4. Backup seed phrase
      </Tab>
      <Tab title="Torus">
        1. Visit [tor.us](https://tor.us/)
        2. Sign up with email/social
        3. No seed phrase needed
      </Tab>
    </Tabs>
  </Step>

  <Step title="Switch to Devnet">
    <Note>
      **Devnet** is Solana's test network. Use it for free testing with fake tokens!
    </Note>

    **For Phantom:**
    1. Open Phantom wallet
    2. Click âš™ï¸ (Settings)
    3. Developer Settings â†’ Change Network
    4. Select "Devnet"
  </Step>

  <Step title="Get Test Tokens">
    You need two types of tokens:

    ### Get Test SOL (for transaction fees)
    <CodeGroup>
    ```bash Using Faucet Website
    # Visit the official Solana faucet
    https://faucet.solana.com/

    # Paste your wallet address
    # Click "Airdrop 1 SOL"
    # Wait 5-10 seconds
    ```

    ```bash Using CLI (Optional)
    solana airdrop 1 <YOUR_WALLET_ADDRESS> --url devnet
    ```
    </CodeGroup>

    ### Get Test USDC
    <Info>
      Detailed instructions: [Wallet Setup Guide](/wallet-setup)
    </Info>

    Quick method:
    1. Use a devnet USDC faucet
    2. Or run our setup script:
    ```bash
    ./scripts/setup-test-usdc.sh
    ```
  </Step>

  <Step title="Connect & Play">
    1. Visit [x402-pixel-war.vercel.app](https://x402-pixel-war.vercel.app)
    2. Click "Connect Wallet"
    3. Select your wallet (Phantom/Solflare)
    4. Approve the connection
    5. Start conquering pixels! ğŸ®
  </Step>
</Steps>

### Method 2: Batch Conquest

<Warning>
  Batch operations are more efficient for creating pixel art!
</Warning>

<Steps>
  <Step title="Enter Selection Mode">
    Hold down `Shift` key
  </Step>
  <Step title="Select Pixels">
    **Option A**: Click multiple pixels
    **Option B**: Click and drag to select an area
  </Step>
  <Step title="Open Batch Modal">
    Click "Batch Conquer" button at the bottom
  </Step>
  <Step title="Review & Confirm">
    The modal shows:
    - Total pixels selected
    - Pixels you can conquer (excluding owned)
    - Total cost
  </Step>
  <Step title="Choose Color & Conquer">
    Pick a color and click "Confirm"
  </Step>
</Steps>

  </Accordion>

  <Accordion icon="palette" title="Create Simple Patterns">

</AccordionGroup>

## Common Issues

<AccordionGroup>
  <Accordion icon="circle-exclamation" title="Wallet Won't Connect">
    **Solutions:**
    1. Refresh the page
    2. Check you're on Devnet
    3. Disable conflicting extensions
    4. Try a different browser
  </Accordion>

  <Accordion icon="circle-exclamation" title="Balance Shows 0">
    **Solutions:**
    1. Verify you have USDC (not just SOL)
    2. Check you're on the correct network
    3. Wait a few seconds for balance to load
    4. Click the balance to refresh manually
  </Accordion>

  <Accordion icon="circle-exclamation" title="Transaction Failed">
    **Common reasons:**
    - Insufficient balance
    - Network congestion
    - Wallet rejected transaction

    **Solution:** Try again or check your balance
  </Accordion>

  <Accordion icon="circle-exclamation" title="Grid Not Loading">
    **Solutions:**
    1. Check your internet connection
    2. Refresh the page
    3. Clear browser cache
    4. Check browser console for errors
  </Accordion>
</AccordionGroup>
